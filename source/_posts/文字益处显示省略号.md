&emsp;&emsp;**1，单行文字溢出显示省略号**
```
width:100px;//需要给个宽度
overflow: hidden;
text-overflow: ellipsis;//文字溢出显示省略号
white-space: nowrap;
```
&emsp;&emsp;**2，多行文字溢出显示省略号（支持webkit内核浏览器）**
```
display: -webkit-box;
overflow: hidden;
text-overflow: ellipsis;
word-wrap: break-word;
white-space: normal !important;
-webkit-line-clamp: 2;//多少行溢出
-webkit-box-orient: vertical;
```
&emsp;&emsp;**3，js实现**
```
//利用scrollHeight和offsetHeigt属性，如果实际高度一直大于自身高度，就一直截取从后往前截取字符，直到内容不超出的时候再手动加上"..."
<div id="a">sdfaaaaaaaaasdfaaaaaaaaasdfaaaaaaaaasdfaaaaaaaaa</div>

<script>
	var a=document.getElementById("a");
	while(a.scrollHeight>a.offsetHeight){
		//scrollHeight包括滚动条的高度即指的是实际内容的高度   offsetHeigt指的是盒子的高度
		a.innerText=a.innerText.substring(0,a.innerText.length-1)
		if(a.scrollHeight==a.offsetHeight){
			//如果内容不超出了再截取三个字符然后加上三个“...”
			a.innerText=a.innerText.substring(0,a.innerText.length-3)+"..."
		}
	}
</script>
```
